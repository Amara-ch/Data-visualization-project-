# Power BI Data Visualization Project

## 📊 Project Overview
This repository showcases a comprehensive Power BI dashboard designed to transform complex datasets into actionable business intelligence. The project emphasizes clean data modeling, interactive storytelling, and the use of specialized custom visuals to enhance data density and user interpretation.

## 🚀 Key Features
* **Advanced Interactive Slicers:** Implementation of the **Chiclet Slicer** for a high-end, button-style navigation and filtering experience.
* **KPI Performance Tracking:** Integrated **Cylindrical Gauges** to provide clear, vertical progress indicators against defined organizational targets.
* **Categorical Analysis:** Advanced distribution views using the **Donut Chart GMO** for detailed categorical breakdowns.
* **Optimized Data Architecture:** A robust star-schema model designed for high-speed query performance and scalability.
* **Automated ETL:** Built-in Power Query (M) transformations for data cleaning and normalization.

## 🛠️ Tools & Visuals Used
* **Power BI Desktop:** Dashboard design and report authoring.
* **DAX (Data Analysis Expressions):** Custom measures for dynamic calculations.
* **Key Custom Visuals:** * ChicletSlicer
    * CylindricalGauge
    * DonutChartGMO

## 📂 File Description
* `projectdv31octobermain.pbix`: The core Power BI file containing all visualizations, data models, and calculated measures.

## 📝 How to Use
1.  **Prerequisites:** Install the latest version of [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
2.  **Access:** Download and open the `.pbix` file.
3.  **Data Source:** To refresh the data, navigate to **Transform Data** > **Data Source Settings** and point the source to your local data directory.
4.  **Interaction:** Use the interactive slicers to drill down into specific timeframes and categories.
